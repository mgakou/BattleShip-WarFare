Structure du code

Le code source du jeu suit l'architecture MVC. Ainsi, le code est réparti en trois grands fichiers:

Le fichier controleur : controller.py

Ce fichier sert à lancer le jeu et fait le lien entre le modèle et la vue.

Le fichier vue : ihm.py

Ce fichier sert à afficher le jeu (bateaux, grille, boutons, interface, menu) dans la fenêtre crée précédemment par controller.py.

Le fichier modèle : data.py

Ce fichier sert à gérer l'état du jeu (taille de la grille, système de vainqueur, ...) et modifier sa représentation interne.
